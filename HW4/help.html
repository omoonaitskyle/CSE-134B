      • General



    1. Frequently asked questions that give you more information on this quiz maker! To return to the quiz maker use the top right button. To navigate to a certain section in the FAQ click on the left menu items.

      2. How to add a question



    2. Users will be able to add the following type of questions using the button insertion tool the top left corner:


      • Multiple Choice
        • Check boxes for correct answers(Ability to have more than one answer)
        • Can delete an answer or add more
      • True/False
      • Fill in the Blank
      • Short Answer

      3. Deleting



    3. In order to delete questions, a trashcan icon is located at the top right corner of each question. Simple click on it and the question will be deleted.

      4. Point Value



    4. Each question can be assigned a point value. Each question begins with 0 points and will not count towards the total allocated points unless the user changes the value.

      5. Reordering Questions



    5. Click on the top right corner drag button a question before or after another question to reorder the questions

      6. Save and Preivew



      • Save: To save your quiz simply click on the Update button towards the bottom left corner.
      • Preview:To view what the quiz looks like click on the preview quiz towards the top.

      7. Timer



    7. Ability to set the quiz as a timed test. The timer is 00:00:00 and goes form a hour, minute, second format. Simple click on the numbers and insert the desired time.

      8. Keyboard Shortcuts



    8. To speed up the quiz making process, users can use the following hot-keys:

      • CTRL + ALT + M: Inserts a multiple choice question at the bottom of the page.
      • CTRL + ALT + T: Insert a true/false question at the bottom of the page.
      • CTRL + ALT + F: Inserts a Fill in the Blank question at the bottom of the page.
      • CTRL + ALT + W: Inserts a Short Answer question at the bottom of the page.
      • CTRL + ALT + P: Allows a user to preview the quiz that they are working on.
      • CTRL + ALT + S: Allows a user to save the quiz that they are working on.
